# Портфолио сайт

Статический сайт-портфолио по макету Figma.

## Как открыть

Локальный сервер уже можно запускать из папки проекта:

```bash
python3 -m http.server 4173
```

После запуска открыть:

```text
http://127.0.0.1:4173
```

Также сайт можно открыть напрямую через `index.html`, но через локальный сервер поведение ближе к обычному сайту.

## Что внутри

- `index.html` — структура страницы и контент.
- `styles.css` — вся адаптивная верстка под desktop / tablet / mobile.
- `assets/*.jpg` — сжатые изображения, которые реально используются на сайте.
- `assets/*.png` — исходные изображения из Figma, оставлены как резерв.

## Что стоит заменить перед публикацией

- Ссылки `Behance` и `in$t@gr@m` сейчас стоят как заглушки.
- При необходимости заменить `tagirvoa@yandex.ru` и телефон на финальные контакты.
- Если нужен деплой, проект можно публиковать как обычный статический сайт.
